# 🍦 Max Sorvetes Ibertioga

Site institucional desenvolvido para negócio local do setor alimentício, com foco em presença digital, SEO local e conversão de clientes via WhatsApp.

## 🚀 Funcionalidades

- Página institucional moderna e responsiva (mobile-first)
- Integração com WhatsApp para contato direto
- Integração com Google Maps para localização da loja
- Cardápio digital organizado para navegação simples
- Estrutura otimizada para SEO local

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
